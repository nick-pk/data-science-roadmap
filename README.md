# 🚀 Career Roadmap: From Java Developer to Data Scientist

## 📌 Overview
This repository documents my 12-month journey transitioning from senior Java developer to data scientist using only free resources.  
Each stage includes a project to demonstrate applied skills, building a portfolio that anyone can follow and replicate.

---

## 🎯 Learning Path
- **Months 1–3**: Python & Basics  
- **Months 4–6**: Statistics & Visualization  
- **Months 7–9**: Machine Learning Fundamentals  
- **Months 10–12**: Advanced ML & AI  
- **Months 13–15**: MLOps & Deployment  
- **Months 16–18**: Domain Specialization  

---

## 📂 Portfolio Projects
1. **Data Cleaning Script** → Python + Pandas  
2. **Visualization Dashboard** → Matplotlib/Seaborn  
3. **Classification Model** → Loan approval prediction  
4. **Sentiment Analysis** → NLP on text data  
5. **Deployed ML Web App** → Flask + Docker  
6. **Domain-Specific Predictive Model** → Healthcare/Finance

## 🚀 How to Use
Each project folder contains:
- **Code** → Jupyter notebooks or Python scripts  
- **Dataset** → Link to source dataset (Kaggle, UCI, etc.)  
- **Report.md** → Documentation (objective, steps, results, future work)  
- **Visuals** → Graphs, screenshots, dashboards
